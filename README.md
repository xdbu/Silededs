# 👤 Silededs - Fake Profile / Identity Generator

Generate fictional French profiles with avatars, realistics bios and complete personal information in just a few seconds.

This tool generates:

* 🇫🇷 French first & last names
* 🏠 Realistic addresses
* 📱 Phone numbers
* 📧 Temporary working email addresses
* 🎂 Birth dates
* ✍️ Realistics bios
* 📸 Profile pictures
* 💾 CSV & JSON exports

> ⚠️ All generated profiles are fictional and intended for development, testing and educational purposes only.

## 📦 Requirements

* Python 3.9+
* Requests
* Pillow
* Faker
* Colorama

Install everything with:

```bash
pip install -r requirements.txt
```

## 📝 Usage

Run the script:

```bash
python main.py
```

1. Generate your bios (they are saved as `bios.txt` in the `input` folder)
2. Generate your profiles
3. Your profiles will automatically be exported to **CSV** and **JSON**, and avatars will be downloaded into the `avatars` folder (they are saved in the output folder).
